# 700. Search in a Binary Search Tree

## Solution 1 -- Recursion

* Time Complexity:
  * Worst Case: O(N), where N is the number of the total nodes in the tree.
  * Average Case: O(log N)

* Space Complexity:
  * Worst Case: O(N)
  * Average Case: O(log N)

## Solution 2 -- Iteration

* Time Complexity:
  * Worst Case: O(N), where N is the number of the total nodes in the tree.
  * Average Case: O(log N)

* Space Complexity: O(1)
