# 322. Coin Change

## Solution 0 -- Brute Force

* Time Complexity: O(N^M), where N is the number of coins and M is the amount.

* Space Complexity: O(M), used for DP array.

## Solution 1 -- DP (Bottom Up)

* Time Complexity: O(N*M), where N is the number of coins and M is the amount.

* Space Complexity: O(M), used for DP array.
