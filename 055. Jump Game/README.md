# 055. Jump Game

## Solution 1 -- Greedy

* Time Complexity: O(N)

* Space Complexity: O(1)

## Solution 2 -- Backtracking (TLE)

* Time Complexity: O(2^N)

* Space Complexity: O(N)

## Solution 3 -- Top-Down Dynamic Programming

* Time Complexity: O(N^2)

* Space Complexity: O(N)

## Solution 4 -- Bottom-Up Dynamic Programming

* Time Complexity: O(N^2)

* Space Complexity: O(N)
